== Description ==

Quick Demo Import is a powerful free WordPress plugin, that is designed especially for theme authors. With the help of this plugin, theme authors can let their users import demo content and preview demos right from the dashboard.

For this plugin to work, themes should be compatible with this plugin. Theme authors can easily make themes compatible with this plugin in a matter of few minutes. [Follow this guide to integrate Quick Demo Import plugin](https://quickdemoimport.com/#integration-guide)

== Installation ==

1. In your admin panel, go to Appearance -> Plugins and click the 'Add New' button.
2. Type in Quick Demo Import in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Install' button.
4. After install is complete click on 'Activate' button.
